Shortcut macro to create an instance of a Present delta type, based on a numeric value and a unit. 

Possible units and their returned types\-:


\begin{DoxyItemize}
\item {\itshape year}, {\itshape years}, {\itshape yr}, {\itshape yrs}\-: \hyperlink{structMonthDelta}{Month\-Delta}
\item {\itshape month}, {\itshape months}\-: \hyperlink{structMonthDelta}{Month\-Delta}
\item {\itshape week}, {\itshape weeks}\-: \hyperlink{structDayDelta}{Day\-Delta}
\item {\itshape day}, {\itshape days}\-: \hyperlink{structDayDelta}{Day\-Delta}
\item {\itshape week\-\_\-time}, {\itshape weeks\-\_\-time}\-: \hyperlink{structTimeDelta}{Time\-Delta}
\item {\itshape day\-\_\-time}, {\itshape days\-\_\-time}\-: \hyperlink{structTimeDelta}{Time\-Delta}
\item {\itshape hour}, {\itshape hours}, {\itshape hr}, {\itshape hrs}\-: \hyperlink{structTimeDelta}{Time\-Delta}
\item {\itshape minute}, {\itshape minutes}, {\itshape min}, {\itshape mins}\-: \hyperlink{structTimeDelta}{Time\-Delta}
\item {\itshape second}, {\itshape seconds}, {\itshape sec}, {\itshape secs}\-: \hyperlink{structTimeDelta}{Time\-Delta}
\item {\itshape millisecond}, {\itshape milliseconds}, {\itshape ms}\-: \hyperlink{structTimeDelta}{Time\-Delta}
\item {\itshape microsecond}, {\itshape microseconds}, {\itshape us}\-: \hyperlink{structTimeDelta}{Time\-Delta}
\item {\itshape nanosecond}, {\itshape nanoseconds}, {\itshape ns}\-: \hyperlink{structTimeDelta}{Time\-Delta}
\end{DoxyItemize}

Note that {\itshape day} and {\itshape week} return a \hyperlink{structDayDelta}{Day\-Delta}, not a \hyperlink{structTimeDelta}{Time\-Delta}. If you need a \hyperlink{structTimeDelta}{Time\-Delta} instead in one of these situations, you need to use {\itshape day\-\_\-time} and {\itshape week\-\_\-time}.

Examples\-:


\begin{DoxyCode}
\hyperlink{structMonthDelta}{MonthDelta} d = \hyperlink{present_8h_a381fc9ebf9765d2a1c1a6b809f39ed6c}{present\_delta}(6, months);
\hyperlink{structDayDelta}{DayDelta}   d = \hyperlink{present_8h_a381fc9ebf9765d2a1c1a6b809f39ed6c}{present\_delta}(-10, days);
\hyperlink{structTimeDelta}{TimeDelta}  d = \hyperlink{present_8h_a381fc9ebf9765d2a1c1a6b809f39ed6c}{present\_delta}(4, weeks\_time);
\hyperlink{structTimeDelta}{TimeDelta}  d = \hyperlink{present_8h_a381fc9ebf9765d2a1c1a6b809f39ed6c}{present\_delta}(1, second);
\end{DoxyCode}
